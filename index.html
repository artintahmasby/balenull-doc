مرجع رسمی و کامل BaleNull

این سند، ترکیب کامل دو نسخه داکیومنت BaleNull است و با هدف حفظ همه قابلیت‌ها، همه متدها، همه مثال‌ها و همه توضیحات تهیه شده است. در این نسخه، هم مدیریت Session، هم متدهای ربات و هم متدهای کلاینت کاربری به‌صورت یکپارچه و دسته‌بندی‌شده ارائه شده‌اند.

فصل ۱: مفاهیم پایه

API Token
توکن اصلی ربات برای ارتباط با API بله و ارسال درخواست‌ها.
Long Polling
روش دریافت آپدیت‌ها با `get_updates` و حلقه `run`.
Dispatcher & Handlers
سیستم ثبت هندلرها برای پردازش پیام‌ها و رویدادها.
Session (.bale)
فایل سشن رمز/کدگذاری‌شده برای ذخیره‌ی اطلاعات ورود و احراز هویت.
Asyncio
پایه اجرای ناهمگام برای ارتباط سریع و غیرمسدودکننده با API.

فصل ۳: ۸ سورس طلایی

1) Echo Bot
پاسخ‌گویی خودکار به هر پیام با همان متن.
2) Welcome Bot
خوش‌آمدگویی به اعضای جدید گروه.
3) Anti-Badword
فیلترکردن کلمات نامناسب.
4) Downloader
دانلود فایل‌ها و ذخیره روی دستگاه.
5) Reaction Bot
افزودن واکنش خودکار به پیام‌ها.
6) Forward Bot
فوروارد خودکار پیام‌ها به مقصد دیگر.
7) Status Bot
مدیریت آنلاین/آفلاین کاربر.
8) Auto-Report Bot
ریپورت خودکار پیام‌های هدف.
# نمونه‌ی Echo Bot from balenull import BaleBot bot = BaleBot("YOUR_BOT_TOKEN") @bot.on_message() async def handler(msg): await msg.reply(msg.text or "Empty") bot.run()

نکات پیشرفته

- برای پروژه‌های بزرگ، هندلرها را ماژولار نگه دارید.
- سشن را امن نگه دارید و فایل `.bale` را در نسخه‌ کنترل قرار ندهید.
- در متدهای فایل، خطاهای شبکه و MIME type را کنترل کنید.
- برای عملکرد بهتر، از `asyncio` و صف پردازش رویدادها استفاده>
